About AI-Powered API Builder

What is it?

The AI-Powered API Builder is a full-stack application that leverages local language models (via Ollama) to automatically generate FastAPI code from natural language descriptions.

How does it work?

  1. You write a prompt - Describe the API you want to build
  2. AI understands your request - The system analyzes your prompt
  3. Code is generated - A complete FastAPI project is created
  4. You review the code - View, edit, and run the generated API

Key Features

  • ✨ Automatic API code generation
  • 🔐 JWT authentication support
  • 📊 CRUD operations scaffolding
  • 🗄️ Database integration with SQLAlchemy
  • ✅ Code validation and security checks
  • 📝 Interactive code editor
  • ▶️ Run generated APIs directly

Tech Stack

Backend

FastAPI • Python 3.11+ • Uvicorn

Frontend

Jinja2 • HTML • CSS

AI Engine

Ollama • Qwen 2.5 Coder 7B

Getting Started

Prerequisites

  • Python 3.11 or higher
  • Ollama installed and running
  • qwen2.5-coder:7b model (or compatible)

Installation


# Clone or download the project
# Install dependencies
pip install -r requirements.txt

# Start Ollama (in another terminal)
ollama serve

# Pull the model (first time only)
ollama pull qwen2.5-coder:7b

# Run the application
cd app
uvicorn main:app --reload

Example Usage

Try generating an API with the following prompt:

"Create a blog API with posts and comments. Include JWT authentication, CRUD operations for posts and comments, and user management."

About the LLM

This application uses Ollama to run language models locally on your machine. It supports models like Qwen 2.5 Coder which are optimized for code generation.

Learn more about Ollama →

Limitations

  • Generated code should be reviewed before production use
  • Complex requirements may need manual refinement
  • Performance depends on your hardware and LLM model size
  • Generated projects are stateless and not persisted by default
